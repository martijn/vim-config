Install with

    curl https://raw.githubusercontent.com/martijn/vim-config/master/bootstrap.sh | sh

Tested with:

* Macvim 7.4
* FreeBSD 9 vim package
* CentOS 5, 6 stock Vim (with minor breakage)
* CentOS 7 stock Vim
