Install with

    curl https://raw.github.com/martijn/vim-config/master/bootstrap.sh | sh

Tested with:

* Macvim 7.3
* FreeBSD 9 vim package
* Linux Mint 12 stock Vim
* CentOS 5 stock Vim (although NERDCommenter currently appears broken)
